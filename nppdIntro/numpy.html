How can we refer to the data store in array?

The graph below address the concept that each element in array has a corresponding index, which allow as to access the data stored in array through the efficient. From the left-most graph, we can see that the index of the first number is zero, and index will increase 1 for each element after. Thus, with data[0] we can get the first element, and data[1] for the seconde.

In the situation we want to access more than one data continuously,we will use data[0:2] and data[1:], which corresponding to get all the elmement with index large or equal to 0 and smaller than 2, get all the data with index larger than 1, respectively.

Moreover, in some special cases, we want to access data in reverse order. In contrast to the normal order, in reverser order, the last element stored is idnexed as -1 and will decrease 1 for each element before. Thus, to get the last 2 element we can use data[-2:] -- get all the data with index larger or equal to -2, which is index -2 and index -1.
