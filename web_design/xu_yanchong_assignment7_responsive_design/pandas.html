View the Data stored in a pd.DataFrame

A data frame is consist of multiple Series as columns or rows. Thus, DataFrame actually can be viewed as a table that can be referenced by either the first row or the first column. The image below shows the way to display the first or last several rows. Through, df.head(), by default, the first 5 rows of the DataFrame is printed.
