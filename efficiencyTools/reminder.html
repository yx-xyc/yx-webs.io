Tags

Tag is actually a new feature provided after system update to IOS 15. Currently, I create 5 tags for my daily usage, including #buy, #eat, #wish, #nyu, and #gre. #nyu are the tags for all my nyu homework, so as #gre. #buy and #eat are just the tags of things related to eat and buy, could be medication for example. #wish are some goals.

With these tags, Reminder can automatically generate list that contains all tasks with the corresponding tags. For example, my NYU Academy list is the list contains all the homework and their DDL. Instead of dragging tasks into the corresponding folders, now what need to be done is just give the correct tags when creating the task.

Task Folders

With these tags, I only need two lists manage manually, which are Life and work. All the tasks would belong either one of them. To further help me to make decision about which task to do, the work list is divided into four sub-lists: emergent and important, emergent but not important, not emergent but important, not emergent and not important.
